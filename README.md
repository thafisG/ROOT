# 🌱 ROOT

> Build better habits, one workout at a time.

ROOT é uma plataforma de bem-estar focada na criação de hábitos saudáveis por meio de recomendações personalizadas.

O principal objetivo do projeto é desenvolver um **Recommendation Engine** capaz de aprender com as interações do usuário e sugerir atividades que incentivem uma rotina mais equilibrada.

> 🚧 Projeto em desenvolvimento.

## 🚀 Tecnologias (previstas)

- Angular
- TypeScript
- SCSS
- PWA
- LocalStorage
